# Лабораторна робота 4: ЕЦП на основі RSA

Цей проєкт показує, як працює електронний цифровий підпис (ЕЦП) на основі алгоритму RSA.

## Що робить програма
- Генерує пару ключів RSA (2048 біт).  
- Підписує текстові та бінарні файли.  
- Перевіряє підпис і показує, якщо файл змінили після підписання.

## Використані технології
Python, PyCryptodome.

## Як запустити
1. Скачай репозиторій.  
2. Запусти `main.py`.  
3. Програма створить ключі та тестові файли, підпише їх і перевірить підпис.  
4. Можна змінити файл і перевірити, що підпис стає недійсним.
